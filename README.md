# Digital-Scoreboard-Controller
VHDL Sports Scoreboard Controller (0-99) for Artix-7 FPGA. Features a robust FSM for score tracking, multiplexed 7-segment display logic, and edge-case handling (simultaneous input inhibition). Includes a 5-cycle fail-safe reset. Fully synthesized and verified in Xilinx Vivado with physical pin constraints on hardware.
